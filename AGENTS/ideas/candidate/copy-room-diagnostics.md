# Copy Room Diagnostics

Status: CANDIDATE
Priority: Low
Last reviewed: 2026-04-22

## Idea

Make it easier to copy/export the most relevant diagnostics for a room.

## Why It Matters

- Could help the user hand off the right logs without guessing.

## User Feedback

- User leaned toward keeping logs as files, possibly categorized, and letting the assistant decide which category matters.

## Risks / Constraints

- Should not replace raw logs entirely.
- Probably best paired with better log categorization if needed.

## Good Entry Points

- current logging paths and debug-playbook
