# Mini Onboarding Inside Plugin

Status: CANDIDATE
Priority: Low
Last reviewed: 2026-04-22

## Idea

Add lightweight help/onboarding that stays discoverable but does not annoy advanced users.

## Why It Matters

- Could make the plugin easier to learn without a big tutorial wall.

## User Feedback

- The user prefers hidden or contextual hints that do not get in the way.

## Risks / Constraints

- Must be skippable and quiet.
- Avoid permanent clutter in settings.

## Good Entry Points

- settings help affordances
- tooltip/help-icon patterns already in the UI
